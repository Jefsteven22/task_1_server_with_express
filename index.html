Ejercicio

1.- Crear un nuevo proyecto de node

2.- Instalar las dependencias necesarias

3.- Crear tus scripts start, dev

4.- Crea un servidor básico con express

5.- Atender las siguietnes peticiones

5.1.- Una petición GET en la ruta '/users' y vas a responder con la siguinte info: 

    {
      "id": "6197f7d7c1b2af0dccecdca2",
      "name": "Juan",
      "lastname":"Perez",
    }

5.2.- Una petición del tipo POST en la ruta '/users', en el body enviar la siguiente info id, name, lastname y responder con el estatus 201 y de la siguietne manera 

    {
      "message": "Usuario creado",
      "userInfo": {informacion del usuario}
    }
